# Demo (optional bonus)

A single static `index.html` — no build step, no bundler. It renders **chart** specs with
[`vega-embed`](https://github.com/vega/vega-embed) and the **network** spec with a small
[`d3-force`](https://d3js.org/) block (both from a CDN), and surfaces **per-datum citations**
on hover (network nodes/edges) and click (chart marks), linking each to its
ClinicalTrials.gov study.

## Run it

```bash
python -m http.server -d demo 8080     # then open http://localhost:8080
```

- **Example browser — fully offline.** The dropdown loads `demo/examples.json` (a bundle of
  the committed example responses, generated by `scripts/capture_examples.py`). No API, no
  key, no network to this backend. It opens on the sponsor↔drug **network** so the deep
  citations are visible immediately — hover a node or edge.
- **Live query box — needs the API.** To ask your own question, also start the backend and
  keep a key in `.env` (live queries plan with the LLM):

  ```bash
  uv run uvicorn app.api.main:app --port 8000 &
  ```

  Permissive CORS is already configured on the API, so the page can call it from `:8080`.

`examples.json` is regenerated whenever you run `scripts/capture_examples.py` (either mode).
